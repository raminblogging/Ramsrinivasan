📋 The Full Flow — Step by Step

        

          
1
Write in admin → Publish. Post saved to localStorage as rs_blogs.

          
2
Blog index auto-updates on the same device. New posts appear at the top immediately.

          
3
Clicking a post opens /blog/post.html?slug=your-slug and renders the full rich content with all your H2/H3 headings.

          
4
Export → commit site-data.json to GitHub after publishing. All visitors on any device will then see the post.

          
5
Old posts are never deleted. Both the blog index and post reader merge localStorage + site-data.json, so every post from both sources always shows up.
